# Boussole

Sovereign EU AI Act compliance assessment agent.

> Boussole is a working name (FR: *compass*). The project is under active build; this README is a placeholder. The full bilingual (FR + EN) README ships with Phase 11. See `CLAUDE.md` for the operating specification.

## Quickstart (development)

```bash
# Install uv if you don't have it
curl -LsSf https://astral.sh/uv/install.sh | sh

# Install dependencies
make install

# Run the Phase 1 smoke test
make smoke
```
